# 검 강화 게임 - 공식 게임 가이드

---

## 1. 게임 소개

**검 강화 게임**은 다양한 등급의 검을 수집하고, 강화하며, 배틀에서 승리하는 모바일 RPG 게임입니다.

### 주요 특징
- **200종 이상의 검** 수집 (6개 등급)
- **강화 시스템** (+1 ~ +30)
- **실시간 PvP 배틀**
- **보스 레이드**
- **시즌 패스 & 업적 시스템**
- **친구 시스템**

## 6. 속성 상성

게임에는 5가지 속성이 있으며, 상성 관계에 따라 **+4% / -4%** 데미지 보정이 적용됩니다.

### 속성 종류
- 🔥 **불** (Fire)
- 💧 **물** (Water)
- 🌿 **자연** (Nature)
- ✨ **빛** (Light)
- 🌑 **암흑** (Dark)

### 상성 관계

```
🔥 불 → 🌿 자연 (불이 풀을 태움)
💧 물 → 🔥 불 (물이 불을 끔)
🌿 자연 → 💧 물 (풀이 물을 흡수)

✨ 빛 ↔ 🌑 암흑 (서로 상극)
```

| 공격 속성 | 유리 (1.04배) | 불리 (0.96배) |
|:--------:|:------------:|:------------:|
| 🔥 불 | 🌿 자연 | 💧 물 |
| 💧 물 | 🔥 불 | 🌿 자연 |
| 🌿 자연 | 💧 물 | 🔥 불 |
| ✨ 빛 | 🌑 암흑 | 🌑 암흑 |
| 🌑 암흑 | ✨ 빛 | ✨ 빛 |

※ 속성 유리 시 약 **+5~6% 승률 우위** 효과

## 7. 스킬 타입 상성

각 검은 고유한 스킬 타입을 보유하며, 타입 간 상성이 있습니다.

### 스킬 타입 종류

| 타입 | 이름 | 설명 |
|:----:|:----:|:----:|
| 🗡️ | 참격 (Slash) | 균형 잡힌 공격 |
| 📍 | 관통 (Pierce) | 방어 관통 |
| 💥 | 폭발 (Blast) | 폭발적 피해 |
| 🩸 | 흡수 (Drain) | 피해량 일부 회복 |
| 🛡️ | 수호 (Guard) | 피해 감소 |

### 상성 관계 (1.10배 / 0.90배)

```
📍 관통 → 🛡️ 수호 (관통이 방어 무시)
🛡️ 수호 → 💥 폭발 (방어가 폭발 흡수)
💥 폭발 → 🗡️ 참격 (범위 공격이 근접 압도)
🗡️ 참격 → 🩸 흡수 (빠른 공격이 흡수 방해)
🩸 흡수 → 📍 관통 (흡수가 관통 피해 회복)
```

## 8. 배틀 시스템

### 전투력 계산

```
전투력 = 기본 ATK + (레벨 × 등급별 레벨 보너스) + 칭호 보너스
HP = 2500 + (전투력 × 1.8)
```
